### 238. 除自身以外数组的乘积
```Rust
impl Solution {
    pub fn product_except_self(mut nums: Vec<i32>) -> Vec<i32> {
        let product = nums.iter().fold(1, |acc, e| acc * e);

        for num in &mut nums {
            *num = product / *num;
        }

        nums
    }
}
```

### 151. 反转字符串中的单词
```Rust
impl Solution {
    pub fn reverse_words(s: String) -> String {
        s.split_whitespace().rev().collect::<Vec<&str>>().join(" ")
    }
}
```
